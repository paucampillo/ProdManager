# Prod Manager App

Prod Manager App es una aplicación para gestionar la producción y los recursos de manera eficiente. Este repositorio contiene el código fuente y las instrucciones para ejecutar el proyecto en tu entorno local.

---

## Requisitos previos

Antes de ejecutar este proyecto, asegúrate de tener instalados los siguientes elementos:

1. **Flutter**: Versión estable más reciente.
   - [Instalar Flutter](https://flutter.dev/docs/get-started/install)
2. **Dart**: Viene incluido con Flutter.
3. **Android Studio o Visual Studio Code**:
   - Android Studio (incluye emulador para probar la app).
   - Visual Studio Code (requiere extensiones de Flutter y Dart).
4. **Dispositivo físico o emulador**:
   - Un dispositivo Android o iOS.
   - Un emulador configurado.
